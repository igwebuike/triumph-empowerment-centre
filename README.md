# Triumph Empowerment Centre Web Platform

Deploy today on Netlify, Vercel, GitHub Pages, or any static host.

## Recommended domain
1. triumphempowerment.org
2. triumphcredit.org
3. tecempowerment.org

## Quick deploy
- Upload this folder to Netlify Drop or Vercel.
- Set the domain DNS to the hosting provider.
- Replace the form placeholder with Tally, Formspree, Airtable, or backend API.
